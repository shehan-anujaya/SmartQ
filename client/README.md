# SmartQ Frontend

React + TypeScript + Redux Toolkit + TailwindCSS frontend for SmartQ application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```env
VITE_API_URL=http://localhost:5000/api
```

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

5. Preview production build:
```bash
npm run preview
```

## Features

- User authentication
- Appointment booking
- Queue management
- Real-time updates
- Responsive design

## Technologies

- React 18
- TypeScript
- Redux Toolkit
- TailwindCSS
- Vite
- React Router
- Axios
